Limited Use Policy

Last Updated: April 2026

Overview

This Limited Use Policy describes how Monkey Block uses sensitive APIs and services that require special disclosure under Chrome Web Store policies. We are committed to using these services only for the specific purposes described below.

Supabase

Purpose: User authentication, Pro account management, and email newsletter management

Data Collected:

  • Email address (only for users who opt in to newsletter or create a Pro account)
  • Session tokens (JWT) stored locally to maintain Pro account login state

How Data Is Used:

  • Email addresses are used solely for sending product newsletters and account-related communications
  • Authentication tokens enable secure access to Pro features
  • No personal data is shared with third parties beyond what is necessary for authentication

Data Sharing:

  • None (data is only shared with Supabase for processing authentication and email delivery)
  • Supabase acts as a data processor only, not a data controller

Data Retention:

  • Email addresses: Until user unsubscribes or deletes their account
  • Authentication tokens: Until user logs out or deletes their account
  • Users can delete their data at any time through the extension settings

Location: EU-based infrastructure (GDPR compliant)

Amplitude Analytics

Purpose: Product analytics and feature usage tracking

Data Collected:

  • Pseudonymous device identifier (browser fingerprint)
  • Feature usage events (e.g., timer started, shield enabled, impulse check triggered)
  • Error events and performance metrics
  • Approximate geographic region (derived from IP at request time)

How Data Is Used:

  • To understand which features are most used
  • To identify and fix bugs
  • To improve user experience
  • To make data-driven product decisions

Data Sharing:

  • None (internal analytics only)
  • Data is NOT used for personalized advertising
  • Data is NOT sold or shared with third parties

Data Retention:

  • Per Amplitude's retention policy (see amplitude.com/privacy)
  • Users can stop all analytics collection by uninstalling the extension

Location: EU servers (GDPR compliant)

ipapi.co

Purpose: Geographic location determination

Data Collected:

  • IP address (sent to ipapi.co for geolocation)

How Data Is Used:

  • To determine approximate country/city location
  • To provide localized experiences
  • To ensure compliance with regional regulations

Data Sharing:

  • None (IP address is sent to ipapi.co but not stored by us)

Data Retention:

  • Real-time only (IP address is not stored by Monkey Block)
  • ipapi.co may retain IP data according to their own policy

Location: Third-party service (check ipapi.co/privacy for details)

Web3Forms

Purpose: Bug report and contact form processing

Data Collected:

  • Email address (provided by user)
  • Message content (provided by user)
  • Optional screenshots (uploaded by user)

How Data Is Used:

  • To process and deliver bug reports to developers
  • To respond to user inquiries

Data Sharing:

  • Web3Forms (form processing service only)
  • Cloudinary (for screenshot storage and processing)

Data Retention:

  • Until the bug report is resolved (typically 30 days)
  • Screenshots are deleted after processing

Location: Third-party services (check web3forms.com/privacy and cloudinary.com/privacy)

Cloudinary

Purpose: Screenshot upload and processing for bug reports

Data Collected:

  • Screenshot images (uploaded by user)

How Data Is Used:

  • To store and process screenshots attached to bug reports
  • To optimize image size for delivery

Data Sharing:

  • None (screenshots are only accessible to Monkey Block developers)

Data Retention:

  • Until the bug report is resolved (typically 30 days)
  • Screenshots are automatically deleted after processing

Location: Third-party service (check cloudinary.com/privacy)

Polar

Purpose: Payment processing for Pro subscriptions

Data Collected:

  • Email address (required for subscription)
  • Billing information (handled directly by Polar; not accessible to Monkey Block)
  • Subscription status

How Data Is Used:

  • To process Pro subscription payments
  • To verify active subscription status
  • To issue receipts and handle refunds

Data Sharing:

  • Only with Polar as the payment processor
  • Monkey Block does not store or access your payment card information

Data Retention:

  • Per Polar's retention policy (see polar.sh/legal/privacy)
  • Subscription data retained as required for tax and accounting compliance

Location: Third-party service (check polar.sh/legal/privacy)

Summary

All sensitive APIs and third-party services are used solely for the specific purposes described above. We do not use these services for:

  • Personalized advertising
  • Selling user data
  • Sharing data with data brokers
  • Cross-site tracking of user behavior

Users can:

  • Stop all analytics collection by uninstalling the extension
  • Unsubscribe from email newsletters at any time
  • Delete all local data by uninstalling the extension
  • Request deletion of data held by third-party services
